Free Hosting Guide

How to Host This Website for Free on GitHub Pages

  1. Create a GitHub account at github.com if you don't have one
  2. Create a new repository named username.github.io (replace "username" with your GitHub username)
  3. Save this HTML file as index.html on your computer
  4. Upload the file to your GitHub repository:
    • Click "Add file" > "Upload files"
    • Drag and drop your index.html file
    • Click "Commit changes"
  5. Go to your repository Settings > Pages
  6. Under Source, select "Deploy from a branch" and choose the main branch
  7. Click Save - your site will be live at https://username.github.io in 1-2 minutes

Alternative free hosting options:

  • Vercel - Drag and drop deployment
  • Netlify - Continuous deployment from Git
  • Cloudflare Pages - Fast static site hosting
